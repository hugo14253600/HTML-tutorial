Explicação da Tag <p>

A tag <p> é usada em HTML para definir um parágrafo. É uma das tags mais básicas e comuns, utilizada para agrupar e formatar textos em blocos de parágrafo.

Exemplo de Uso:

Veja um exemplo simples de como utilizar a tag <p>:

<p>Este é um parágrafo de exemplo.</p>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Este é um parágrafo de exemplo.

Propriedades:

Os parágrafos são elementos de bloco, o que significa que ocupam toda a largura disponível e sempre começam em uma nova linha. Eles também possuem margem superior e inferior padrão para separá-los de outros elementos.

Estilização:

É possível estilizar parágrafos usando CSS. Por exemplo, você pode alterar a cor do texto, a fonte, o espaçamento e muito mais. Veja um exemplo:

p {
    color: blue;
    font-size: 16px;
}

O código CSS acima alteraria a cor do texto dos parágrafos para azul e definiria o tamanho da fonte para 16 pixels.
