Explicação da Tag <meta>

A tag <meta> é usada para definir metadados sobre o documento HTML. Metadados são informações sobre os dados e não são exibidos na página. Eles são tipicamente usados pelos navegadores e mecanismos de busca.

Atributos Comuns:

  • charset: Especifica a codificação de caracteres do documento.
  • name e content: Definem um par de nome/valor para metadados. Comumente usado para descrição, palavras-chave, autor, etc.
  • http-equiv: Fornece cabeçalhos HTTP para o navegador.
  • viewport: Fornece instruções sobre como controlar as dimensões e a escala da janela de visualização.

Exemplo de Uso:

Veja alguns exemplos de como utilizar a tag <meta>:

<head>
<meta charset="UTF-8">
<meta name="description" content="Exemplo de uso da tag meta">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="Seu Nome">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de Documento HTML</title>
</head>

Importância:

Usar a tag <meta> corretamente é crucial para SEO (Search Engine Optimization) e para garantir que seu documento HTML seja exibido corretamente em diferentes dispositivos e navegadores.
