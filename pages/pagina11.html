Explicação da Tag <table>

A tag <table> é usada para criar uma tabela em uma página HTML. Ela consiste em linhas (<tr>), colunas de cabeçalho (<th>) e células de dados (<td>).

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <table>:

Nome Idade Cidade
João 30 São Paulo
Maria 28 Rio de Janeiro
Pedro 35 Curitiba

Atributos Comuns:

A tag <table> pode ter vários atributos, incluindo:

  • border: Define a largura da borda da tabela (geralmente não recomendado, pois é melhor usar CSS).
  • width: Define a largura da tabela.
  • cellpadding: Define o espaçamento interno das células.
  • cellspacing: Define o espaçamento entre as células.

Estilização com CSS:

Você pode estilizar tabelas usando CSS para personalizar sua aparência. Aqui está um exemplo básico de estilização:

table {
    width: 100%;
    border-collapse: collapse;
}

th, td {
    border: 1px solid #ccc;
    padding: 8px;
    text-align: left;
}

th {
    background-color: #f2f2f2;
}

O código acima define uma tabela com bordas, células de cabeçalho cinza claro e células de dados com espaçamento interno.
