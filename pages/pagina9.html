Explicação da Tag <button>

A tag <button> é usada para criar um botão clicável em uma página HTML. Ela pode conter texto, imagens ou outros elementos HTML.

Atributos Comuns:

A tag <button> pode ter vários atributos, incluindo:

  • type: Especifica o tipo de botão. Os valores comuns são submit, reset e button.
  • name: Define um nome para o botão, útil quando enviado em um formulário.
  • value: Define o valor associado ao botão, geralmente usado com formulários.
  • disabled: Desabilita o botão para interações.

Estilização com CSS:

Você pode estilizar um botão usando CSS para personalizar sua aparência. Aqui está um exemplo básico de estilização:

.example-button button {
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
    border-radius: 4px;
    font-size: 16px;
}

.example-button button:hover {
    background-color: #45a049;
}

O código acima define um botão verde com efeito de hover mais escuro.
