Explicação da Tag <ul>

A tag <ul> é usada para definir uma lista não ordenada em HTML. Cada item da lista é definido com a tag <li> (item de lista).

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <ul>:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
    </head>
    <body>
        <h1>Minha Lista de Compras</h1>
        <ul>
            <li>Maçãs</li>
            <li>Bananas</li>
            <li>Laranjas</li>
        </ul>
    </body>
</html>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Minha Lista de Compras

  • Maçãs
  • Bananas
  • Laranjas

Atributos Comuns:

A tag <ul> pode ter alguns atributos comuns, embora sejam menos usados hoje em dia com a adoção de CSS para estilização. Aqui estão alguns:

  • type: Define o tipo de marcador para os itens da lista. Os valores possíveis incluem disc, circle e square. No entanto, é recomendado usar CSS para isso.

Estilização com CSS:

Você pode usar CSS para estilizar a lista. Aqui está um exemplo:

ul {
    list-style-type: square;
    padding-left: 20px;
}

li {
    color: #333;
}

Quando aplicado ao exemplo acima, o CSS resultará em uma lista com marcadores quadrados e texto de cor cinza escuro.
