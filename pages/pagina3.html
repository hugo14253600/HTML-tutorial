Explicação da Tag <div>

A tag <div> é um contêiner genérico usado em HTML para agrupar elementos e aplicar estilos ou scripts a eles. Não tem um significado semântico específico, mas é muito útil para a organização e formatação do conteúdo.

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <div>:

<div>
    <h2>Título do Div</h2>
    <p>Este é um parágrafo dentro de um <div>.</p>
</div>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Título do Div

Este é um parágrafo dentro de um <div>.

Propriedades:

O <div> é um elemento de bloco, o que significa que ocupa toda a largura disponível e sempre começa em uma nova linha. Pode conter outros elementos de bloco ou elementos inline.

Estilização:

É possível estilizar <div> usando CSS. Por exemplo, você pode alterar o fundo, a borda, o espaçamento e muito mais. Veja um exemplo:

div {
    background-color: #f9f9f9;
    border: 1px solid #ccc;
    padding: 15px;
    margin: 10px 0;
}

O código CSS acima alteraria a cor de fundo, a borda, o espaçamento interno e a margem dos elementos <div>.
