Explicação da Tag <br>

A tag <br> é usada para inserir quebras de linha em um texto. Diferente das tags de bloco, ela não cria uma nova linha inteira, mas apenas quebra a linha atual, continuando o texto na próxima linha.

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <br>:

<p>
    Esta é uma linha de texto.<br>
    E esta é a próxima linha.
</p>

Auto-fechamento:

A tag <br> é uma tag vazia, o que significa que não precisa de uma tag de fechamento. Pode ser usada simplesmente como <br>.

Uso Comum:

É comumente usada para:

  • Quebras de linha em endereços ou poemas.
  • Quebras de linha em textos longos para melhorar a legibilidade.
