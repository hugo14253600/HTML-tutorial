Explicação da Tag <head>

A tag <head> é uma seção do documento HTML que contém metadados e links para recursos externos, como arquivos CSS e JavaScript. O conteúdo dentro da tag <head> não é exibido diretamente na página web.

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <head>:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
        <link rel="stylesheet" href="estilos.css">
        <script src="script.js"></script>
    </head>
    <body>
        <h1>Bem-vindo à minha página!</h1>
        <p>Este é um parágrafo dentro da tag <body>.</p>
    </body>
</html>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Bem-vindo à minha página!

Este é um parágrafo dentro da tag <body>.

Elementos Comuns:

A tag <head> geralmente contém os seguintes elementos:

  • <meta>: Define metadados sobre o documento, como a codificação de caracteres e a descrição.
  • <title>: Define o título do documento, que é exibido na aba do navegador.
  • <link>: Usado para incluir arquivos externos, como folhas de estilo CSS.
  • <script>: Usado para incluir arquivos de script, como JavaScript.
  • <style>: Contém CSS interno para estilizar a página.

Propriedades:

A tag <head> não possui atributos comuns, mas os elementos contidos nela frequentemente possuem vários atributos. Por exemplo, o elemento <meta> pode ter um atributo charset para definir a codificação de caracteres.
