Explicação da Tag <script>

A tag <script> é usada para incorporar ou referenciar scripts de JavaScript em um documento HTML. Ela pode ser usada para adicionar funcionalidades dinâmicas e interatividade à página.

Atributos Comuns:

  • src: Especifica o URL do arquivo de script externo.
  • type: Especifica o tipo de script. O valor padrão é "text/javascript".
  • async: Especifica que o script deve ser executado de forma assíncrona.
  • defer: Especifica que o script deve ser executado após o documento ser analisado.

Exemplo de Uso:

Veja alguns exemplos de como utilizar a tag <script>:

<!-- Script interno -->
<script>
console.log('Olá, mundo!');
</script>

<!-- Script externo -->
<script src="script.js"></script>

Localização:

A tag <script> pode ser colocada no <head> ou no <body> do documento HTML. Scripts que afetam a renderização da página geralmente são colocados no final do <body> para não bloquear o carregamento da página.

Importância:

Usar a tag <script> permite adicionar interatividade e funcionalidades dinâmicas às páginas web, tornando-as mais interativas e úteis para os usuários.
