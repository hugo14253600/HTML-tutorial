Explicação da Tag <img>

A tag <img> é usada para incorporar uma imagem em um documento HTML. Esta tag é vazia, ou seja, não possui uma tag de fechamento.

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <img>:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
    </head>
    <body>
        <h1>Minha Imagem Favorita</h1>
        <img src="exemplo.jpg" alt="Descrição da Imagem" width="500" height="300">
    </body>
</html>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Minha Imagem Favorita

Atributos Comuns:

A tag <img> possui vários atributos comuns, incluindo:

  • src: Define a URL da imagem.
  • alt: Fornece um texto alternativo que descreve a imagem. Esse texto é exibido se a imagem não puder ser carregada.
  • width: Define a largura da imagem (em pixels ou porcentagem).
  • height: Define a altura da imagem (em pixels ou porcentagem).
  • title: Fornece um texto adicional que aparece como uma dica quando o usuário passa o mouse sobre a imagem.

Importância do Atributo alt:

O atributo alt é importante por várias razões:

  • Acessibilidade: Descreve a imagem para usuários com deficiências visuais que utilizam leitores de tela.
  • SEO: Ajuda os mecanismos de busca a entenderem o conteúdo da imagem, melhorando a indexação da página.
  • Fallback: Fornece um texto descritivo caso a imagem não possa ser carregada.
