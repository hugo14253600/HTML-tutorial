Explicação da Tag <span>

A tag <span> é usada para agrupar elementos inline e aplicar estilos a partes específicas de um texto ou conteúdo sem semanticamente alterar o significado do conteúdo.

Estilização com CSS:

Você pode estilizar o conteúdo dentro de uma tag <span> usando CSS para personalizar sua aparência. Aqui está um exemplo básico de estilização:

.example-span span {
    color: #dc3545; /* Cor do texto */
    font-weight: bold; /* Texto em negrito */
}

O código acima define o estilo para o conteúdo dentro da tag <span>, neste caso, tornando o texto destacado em vermelho e em negrito.
