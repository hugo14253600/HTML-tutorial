Explicação das Tags <h1> a <h6>

As tags <h1> a <h6> são usadas em HTML para definir cabeçalhos, com <h1> sendo o mais importante (ou maior) e <h6> o menos importante (ou menor). Estas tags são essenciais para estruturar o conteúdo de uma página e para otimização de mecanismos de busca (SEO).

Exemplo de Uso:

Veja um exemplo de como utilizar essas tags:

<h1>Cabeçalho de Nível 1</h1>
<h2>Cabeçalho de Nível 2</h2>
<h3>Cabeçalho de Nível 3</h3>
<h4>Cabeçalho de Nível 4</h4>
<h5>Cabeçalho de Nível 5</h5>
<h6>Cabeçalho de Nível 6</h6>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Cabeçalho de Nível 1

Cabeçalho de Nível 2

Cabeçalho de Nível 3

Cabeçalho de Nível 4

Cabeçalho de Nível 5
Cabeçalho de Nível 6

Propriedades:

Os cabeçalhos são elementos de bloco, o que significa que ocupam toda a largura disponível e sempre começam em uma nova linha. Eles também têm tamanhos de fonte e margens padrão que diminuem de <h1> para <h6>.

Estilização:

É possível estilizar cabeçalhos usando CSS. Por exemplo, você pode alterar a cor do texto, a fonte, o espaçamento e muito mais. Veja um exemplo:

h1 {
    color: red;
    font-size: 2em;
}
h2 {
    color: green;
    font-size: 1.75em;
}
h3 {
    color: blue;
    font-size: 1.5em;
}

O código CSS acima alteraria a cor e o tamanho da fonte dos cabeçalhos <h1>, <h2> e <h3>.
