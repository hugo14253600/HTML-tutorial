Explicação da Tag <video>

A tag <video> é usada para incorporar vídeos em uma página HTML. Ela suporta várias funcionalidades, como controle de reprodução, legendas e múltiplas fontes de vídeo.

Atributos Comuns:

  • src: Define o caminho para o arquivo de vídeo.
  • controls: Adiciona controles de reprodução (play, pause, etc.).
  • autoplay: Faz o vídeo iniciar automaticamente quando a página é carregada.
  • loop: Faz o vídeo repetir após terminar.
  • muted: Inicia o vídeo sem som.
  • poster: Define uma imagem a ser exibida antes de o vídeo começar a tocar.

Compatibilidade:

A tag <video> é suportada pela maioria dos navegadores modernos, mas é uma boa prática fornecer múltiplas fontes de vídeo (por exemplo, MP4, WebM) para garantir compatibilidade ampla.
