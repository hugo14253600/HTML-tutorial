Explicação da Tag <html>

A tag <html> é a raiz de um documento HTML. Ela envolve todo o conteúdo de uma página web, incluindo as seções <head> e <body>.

Exemplo de Uso:

Veja um exemplo de como utilizar a tag <html>:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
    </head>
    <body>
        <h1>Bem-vindo à minha página!</h1>
        <p>Este é um parágrafo dentro da tag <html>.</p>
    </body>
</html>

Quando renderizado em um navegador, o código acima resulta no seguinte:

Bem-vindo à minha página!

Este é um parágrafo dentro da tag <html>.

Propriedades:

A tag <html> possui alguns atributos importantes, como o atributo lang, que define o idioma principal do conteúdo da página. Por exemplo, <html lang="pt-BR"> define o idioma como português brasileiro.

Estrutura do Documento:

Um documento HTML básico geralmente segue a seguinte estrutura:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
    </head>
    <body>
        <!-- Conteúdo visível vai aqui -->
    </body>
</html>
